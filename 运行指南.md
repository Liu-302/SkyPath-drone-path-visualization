# SkyPath 运行指南

## 前置要求

- Java 17+、Maven 3.6+
- Node.js 18+、Yarn
- Docker Desktop（可选，用于本地 MongoDB）

## 本地开发（Docker MongoDB）

### 1. 启动 MongoDB

```bash
docker-compose up -d
```

### 2. 启动后端

```bash
cd backend
mvn spring-boot:run
```

### 3. 启动前端

```bash
cd frontend
yarn install
yarn dev
```

前端默认：`http://localhost:8000`，后端：`http://localhost:8080`。

## 使用 MongoDB Atlas（生产/远程）

**请勿在文档或仓库中填写真实连接串或密码。**

1. 在 [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) 创建集群并获取连接 URI
2. 仅通过环境变量传入，例如（**以下为占位示例，请替换为你的实际值并勿提交到 Git**）：

```bash
set SPRING_PROFILES_ACTIVE=prod
set MONGODB_URI=你的Atlas连接串
mvn spring-boot:run
```

请将 `你的Atlas连接串` 替换为 Atlas 控制台提供的 URI，并确保该值只存在于本机环境或密钥管理工具中。

## 安全说明

- 数据库连接串、JWT 密钥、邮箱/密码等敏感信息请仅放在环境变量或 `.env` 中
- 确保 `.env`、`.env.local`、`.env.prod` 已加入 `.gitignore`
- 若曾误将凭证提交到仓库，请立即在对应服务中修改密码并轮换密钥
